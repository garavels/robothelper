# Injury Detection - Robot Dog Search & Rescue

CV pipeline that detects potentially injured people (lying down) vs OK (standing/sitting) using YOLOv8-pose keypoints and a trained classifier.

## Quick Start

### 1. Train the model (Google Colab)

1. Open `injury_detection.ipynb` in [Google Colab](https://colab.research.google.com/)
2. Set runtime to **GPU (T4)**
3. Run all cells — this generates synthetic pose data, trains the classifier, and exports `injury_classifier.pkl`
4. Download `model/injury_classifier.pkl` and place it in `cv_model/model/`

### 2. Run inference locally

```bash
# Install dependencies
pip install ultralytics scikit-learn opencv-python joblib

# Webcam (live demo)
python inference.py

# Video file
python inference.py --source path/to/video.mp4

# Single image
python inference.py --source path/to/image.jpg
```

Press `q` to quit the live window.

### 3. Model size options

Use `--model-size` to trade speed for accuracy:

| Flag | Model | Speed | Accuracy |
|------|-------|-------|----------|
| `-n` | YOLOv8n-pose | Fastest | Good |
| `-s` | YOLOv8s-pose | Fast (default) | Better |
| `-m` | YOLOv8m-pose | Medium | Best |

## How It Works

1. **YOLOv8-pose** detects people and extracts 17 body keypoints per person
2. **Feature engineering** computes 8 geometric features (body angle, aspect ratio, vertical spread, etc.)
3. **Gradient Boosting classifier** (trained on synthetic pose data) predicts OK vs INJURED
4. Results are drawn on the frame with color-coded bounding boxes (green = OK, red = INJURED)
